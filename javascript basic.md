## Javascript书写位置

- 行内Javascript
- 内部Javascript
- 外部Javascript

### 内部js

直接写在html标签里面，用script标签包住。规范：script标签写在</body>上面

```javascript
alert('你好，Javascript')； //页面弹出警告对话框
```

### 外部js

代码写在以.js结尾的文件里 语法：通过script标签，引入到html页面中。

引入外部js后，script标签内不要再写其他代码，否则会被忽略。

```javascript
<body>
  <script src="my.js"></script>
</body>
```

### 行内js（作为了解即可）

代码直接写在标签内部。 语法：直接在标签内部写js代码。

```javascript
<body>
  <button onclick="alert('逗你玩~~~')">点击我月薪过万</button>
</body>
```

## 变量

计算机用来存储数据的盒子，记录不同类型的数据。

### 变量的声明和赋值

数字直接进行存储；字符用单引号引起来表示一段信息；

```javascript
let 变量名
let age //age即变量的名称，也为标识符
let age = 18 //可以在声明变量的同时直接对变量进行赋值（变量的初始化）
```

通过变量名来得到变量里面的数据。

### 声明多个变量

多个变量中间用逗号隔开（不提倡这样去做）

```javascript
let age = 18, uname = 'pink'
```

### 变量的本质

**内存**：计算机存储数据的地方，相当于一个空间

**变量本质**：程序在内存中申请一块用来存放数据的**小空间**

### 变量的命名规则和规范

规则：必须遵守，不遵守就会报错

- 不能用关键字
- 只能用下划线、字母、数字、还有💲，数字不能开头
- 严格区分大小写

规范：建议，以遵守不会报错，但不符合业内通识

- 起名有意义
- 小驼峰命名法 userName

### 常量

**和变量的区别**：常量里面保存的值是不允许改变的。为了程序的安全

声明常量的时候必须直接赋值（初始化），不能先声明之后再赋值

```javascript
const pi = 3.14 //声明常量
console.log(pi) //const 是常量的意思
```



## 数据类型

js数据类型分为基本数据类型、引用数据类型两大类

### 数字类型

可以用于运算操作，对数字进行加减乘除操作。将整数、小数统一称为数字类型

```javascript
console.log(typeof num1) //typeof 可以用来检测数据类型
```

### 字符串类型（string）

被引号包裹的一段文字信息。通过单引号、双引号、反引号包裹的字符串都属于字符串，推荐使用单引号

为了解决字符串嵌套问题，可以单双号交替使用

### 模板字符串

将字符串和变量拼接到一起，使用加号。（较为麻烦）

语法：

```javascript
console.log(`pink老师今年${age}岁`) //反引号中间变量套，直接$大括号
```

反引号中的字符串是可以进行换行的，避免超长字符串阅读困难

### 布尔类型（boolean）

用于判断真假的数据类型，用来判断条件是否成立

有两个固定值 true 和 false

### 未定义类型（undefined）

只声明变量，不赋值的情况下，变量的默认值就为undefined。在开发中经常等待传送过来的数据，此时可以判断undefined来判断数据是否已经传递进来，然后再用变量处理数据

### 空类型

null表示内容为空，返回的结果是object 是一个对象

## 类型转换

把一种数据类型转换成另一种数据类型。

使用需求：表单、prompt获取过来的数据默认是字符串类型的，此时就不能直接进行简单的加法运算

### 显示转换

自己手动写代码告诉系统该转换成什么数据类型。类型转换主要转换成数字型、字符串型、布尔型、

1.转换成数字型

```javascript
Number('1')
//下面两种拥有固定使用场景 ,如果无法完成转换则返回Nan
parseInt('100px') //答案输出100
parseFloat('100.5px') //保留小数 答案输出100.5
```

2.转换成字符型

```javascript
console.log(String(1)) //将数字1转换成了字符串类型 更推荐使用
//变量.toString
console.log(num.toString(可以写进制))
```

3.转换成布尔类型

```javascript
//这六种情况全会转换成false ，其余全部会转换成true

0
''
null
undefined
Nan
```

### 隐式转换

某些运算符被执行时，系统内部自动将数据类型进行转换，进行隐式转换

1.隐式转换为数字型

```javascript
// 这些符号- * / > == +(正号)
console.log(8 - '3') //答案是5
```

2.转换成布尔类型运算符

```javascript
// !
console.log(!true) //false
console.log(!0) //true
```

3.转换成字符串类型运算符

```javascript
// +
console.log('pink' + 18) //最后输出字符串类型的18
```

## 流程控制语句

表达式：可以被求值的代码

语句：一段可以执行的代码，是一个行为

### 分支语句

```javascript
if (条件) { //小括号内的条件结果是布尔值，结果是true则执行大括号内的语句
  满足条件要执行的代码
}
//小括号内的值不是布尔值则会转成布尔值


if (条件) {
  满足条件执行的代码
} else {
  不满足条件执行的代码
}

//三元运算符
//一些简单的双分支语句可以使用三元运算符
条件 ？ 表达式1 : 表达式2
//条件为真，则执行表达式1，否则执行表达式2

//switch分支语句
switch(表达式) {
    case 值1：
    代码1
    break
    
    case 值2：
    代码2
    break
    
    default
    代码n
}
```

### 循环语句

重复执行指定的一段代码，比如我们想输出10次一个语句

```javascript
while (循环条件) {
  要重复执行的代码(循环体)
}

//for循环
for(初始值；循环条件；变量计数) {
  代码体
}
    
// break 中止整个for循环，一般用于结果已经得到，后面循环不需要的时候
// continue中止本次循环，一般用于排除或者跳过某一个选项
```

无限循环

```javascript
while(true) //造成无限循环
for(;;) //造成无限循环
```

## 数组

> 数组里面的索引号从0开始

数组是一种数据类型，属于引用数据类型。可以在单个变量名下存储多个数据（任意类型均可）。

声明语法

```javascript
let 数组名 = [数据1,数据2,... ,数据n]‘
//取值语法
数组名[下标]
//获取数组长度
数组.length
```

新增元素：

通过 数组.push 的方法将一个或者多个元素添加到数组的末尾（重点）

返回值：该数组的新长度，会修改原数组

通过 数组.unshift 将一个或多个元素添加到数组的开头

返回值：该数组的新长度，会修改原数组

```javascript
let arr = ['迪丽热巴']
arr.push('佟丽丫丫')
arr.unshift('古利札那')
```

删除元素：

通过 数组.pop() 从数组中删除最后一个元素

通过 数组.shift() 从数组中删除第一个元素

```javascript
let arr = ['red','green']
console.log(arr.pop())
console.log(arr.shift())
```

**数组.splice() **方法 可以添加或者删除数组元素

```javascript
splice(start,deleteCount) 删除操作
start 起始位置：指定修改开始的位置
deleteCount 删除个数：要删除的元素个数

增加操作
splice(start,deleteCount,item1...)
//在新增时，第二个删除个数写0
```

## 函数

**函数**是可以被重复使用的代码块，将计算的逻辑封装到函数，进行代码复用

### 定义函数

```javascript
//利用function关键字定义函数（声明函数）
function 函数名() {
  函数体
}
```

### 调用函数

```javascript
//调用函数
函数名()
```

### 默认参数

可以给形参一个默认值，在没有传入参数的时候，函数就会使用给到的默认值

```javascript
function sum(x = 0, y =0)
{
  return x + y
}
```

### 函数封装-数组求和

```javascript
function sum(arr) {
  let sum = 0
  for(let i = 0; i < arr.length; i++)
    {
      sum += arr[i]
    }
  return sum
}
sum([1,2,3])
```

### 返回值

返回值：把处理结果返回给调用者 （立即结束函数，后面代码不会被执行）

### 匿名函数

没有名字的函数（必须先声明后使用）

**函数表达式**：将匿名函数赋值给一个变量，并通过变量名称对函数进行调用

```javascript
btn.onclick = function() {
  slert('匿名函数')
}
```

**立即执行函数IIFE**：

避免全局变量之间的污染

```javascript
//方式1
(匿名函数)();
//方式2
(匿名函数());
//不需要调用立即执行
```



## 作用域

**作用域**：变量或者值在代码中可用性的范围，他增强了程序逻辑的局部性，增强了程序的可靠性，减少了冲突

**全局作用域**：（全部变量在任何区域都可以访问或修改）在script标签的内部或者单独的js文件

**局部作用域**：（只能在当前局部内部进行访问或者修改）函数作用域或者块级作用域（在大括号内）

**访问原则**：变量访问原则是就近原则，先找局部再找全局。

## 对象

js中的一种数据类型（引用类型），用于存储数据

**好处**：可以用来详细的描述某个事物，使用键值对形式存储，语义更加明了。数据是无序的

```javascript
//声明方式示例
let pig = {
  //放属性 用冒号，多个属性用逗号隔开
  uname: '佩奇',
  age: 4,
  sex: '女',
  weight: 12.6
  //放方法
  sing: function() {
    console.log('哼~')
  }
}
```

访问对象的属性：

```javascript
pig.age //可以拿到对象对应的属性
```

增加对象的内容：

```javascript
对象.新属性 = 新值
对象.新方法名 = function(){}
```

删除对象内容：

```javascript
delete 对象名.属性名
```

特殊属性：

```javascript
let pig = {
  'pig-name' : '佩奇'
}
console.log(pig['pig-name']) //使用中括号的形式获取到属性
```

遍历对象：

对象没有长度，对象是无序的，使用for in 遍历对象

```javascript
for (let 变量 in 对象) {
  console.log(变量)  //属性名
  console.log(对象[变量]) //属性值
}
```

## 数据类型的存储

内存中堆栈空间分配区别：

栈：优点访问速度块，基本数据类型存放到栈里面

堆：优点存储容量大，引用数据类型（数组、函数、对象）存放到堆里面

引用数据类型将地址存放在栈里面
